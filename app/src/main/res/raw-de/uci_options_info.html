Dies ist ein Editor mit dem man UCI-Optionen, die von einer Engine definierte wurden, ändern kann.

Die Engine gibt für jede Option einen Standard-Wert vor, den man hier, in einem Editier-Feld, ändern kann. Der Optionname wird entweder in gelber Textfarbe (Standard) oder weiß (Änderung) angezeigt. Bei Berührung des Optionnamens wird ein Änderungswert auf den Standard-Wert zurückgesetzt.

Tastensteuerung:
Info: diese Dokumentation
Abbrechen: die gespeicherten Daten bleiben erhalten
Zurücksetzen: alle Optionen werden auf den Standard-Wert zurückgesetzt
OK: die Optionen werden gespeichert und sind nach dem nächsten Start der Engine wirksam

Manche Engines bieten die Möglichkeit an, mit Zusatzdateien (z.B. nnue, weights), die Spielstärke der Engine zu verbessern.
Ist im Optionname der Text "file" enthalten, dann kann man eine Zusatzdatei, wie folgt, aktivieren:
1) mit einem Dateimanager die Datei in ein Unterverzeichnis (oder neu erstelltes Verzeichnis) von: Android / data / ccc.chess.gui.chessforall / files / engines
kopieren
2) im Editierfeld der Option die zuvor kopierte Datei auswählen
3) die OK-Taste drücken

Eine genaue Beschreibung der UCI-Optionen findet man in der Dokumentation der jeweiligen Schach-Engine.
