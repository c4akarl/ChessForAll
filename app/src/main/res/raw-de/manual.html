PGN-Dateien

Folgende Funktionen stehen zur Verfügung:

  • Partie laden
  • Partie speichern, einfügen, ersetzen und löschen
  • PGN-Datei downloaden
  • Partie in die Zwischenablage
  • Position in die Zwischenablage
  • Partie aus der Zwischenablage

Durch langes Drücken auf eine PGN-Datei wird diese (.pgn) und die Datenbankdatei (.pgn-db) gelöscht.

Eine PGN-Datei kann auch von einer Dateimanager-App geladen werden.

Datenbank (.pgn-db)

Das Programm verwendet eine eigene Schachdatenbank (Datei: .pgn-db) zum Verwalten von Partien. Ist beim Laden/Speichern keine Schachdatenbank-Datei vorhanden, wird diese erstellt. In der Statusbar wird der aktuelle Stand der Datenbankerstellung angezeigt. Während dieses Vorgangs ist die PGN-Datei gesperrt. Andere Aktionen können ohne Probleme ausgeführt werden.

Die Verwendung der PGN-Datenbank kann in den Einstellungen deaktiviert werden. Beim Speichern wird dann eine Partie immer an das Dateiende angehängt.

Eröffnungsbücher

Ein kleines Eröffnungsbuch ist in ChessForAll bereits integriert. Es können jedoch auch externe Polyglot-Eröffnungsbücher (.bin) verwendet werden. Die Verwaltung der Eröffnungsbücher erfolgt in den Einstellungen.

Schach-Engines

Aufruf über das Hauptmenü und der Auswahl "Schach-Engine". Folgende Funktionen gibt es:

Schach-Engine auswählen

Mit der Auswahl wird die Haupt-Engine festgelegt.

Diese App verwendet ausschließlich UCI-Engines die mit dem Open Exchange (OEX) Protokoll bereitgestellt werden. Im Play Store gibt es Apps die Schach-Engines im OEX-Format bereitstellen (Suche: "schach engines oex").
Zwei Engines, Rodent und Stockfish (Endung "CfA"), sind in "Chess for All" integriert.

Engine-Optionen

Für die mit "Schach-Engine auswählen" ausgewählte Engine können die vom Engineentwickler festgelegten Optionen geändert werden. Im Änderungsdialog gibt es links unten eine Taste "Info" mit einer genaueren Beschreibung der möglichen Aktivitäten. Die UCI-Optionen werden gespeichert und vor Beginn eines neuen Spiels aktiviert.

Computer-Match

Mit dieser Optionen können mehrere Partien, sowohl im Einzel-Engine als auch im Engine gegen Engine Modus, automatisch ablaufen.

Folgende Funktionen stehen zur Verfügung:
  • Auswahl "Engine / Engine" oder "Enzel-Engine" (Name der Haupt-Engine)
  • Engine Auswahl-Tasten "Weiss", "Schwarz" bzw. "Engine"
  • Auswahl "Aktuelle Partie". Ist diese Option nicht aktiviert, wird jede Partie mit der aktuellen Grundstellung (Standard oder   Chess960) begonnen. Ist diese Option aktiviert, beginnt jede Partie, mit der aktuellen Brettstellung.
  • Auswahl "Farbe wechseln": im Modus "Engine / Engine" wird nach jeder Partie die Farbe (Weiss, Schwarz) gewechselt.
  • Auswahl "Partie speichern". Bei Aktivierung wird jede Partie gespeichert (Pfad, Datei)
  • Pfad, Datei: mit der Taste "PGN" wird der PGN-Dateimanager aufgerufen. Hier kann man Dateien verwalten und auch vorhanden   Dateien auswählen (Computer-Match fortsetzen).
  • Ereignis, Ort. Editierfelder die geändert werden können.
  • Runde: bereits abgeschlossene Partien
  • Partien: maximale Anzahl der Partien. Nach Beendigung der letzten Partie wird der Wettkampf automatisch beendet.
  • Anzeige der Wettkampfdaten: Name der Engines und Ergebnis
  • Taste "Abbrechen": der Vorgang wird abgebrochen
  • Taste "Start / Fortsetzen": ein neuer Wettkampf wird begonnen (Runde: 0) oder ein unterbrochener Wettkampf fortgesetzt.
Nach Betätigung von "Start / Fortsetzen" wechselt die App in den "automatischen Wettkampfmodus" und alle wichtigen Daten des Wettkampfes werden angezeigt. In diesem Zustand können keine weiteren Funktionen aufgerufen werden. Statt dessen kommt ein Computer Match Dialog mit folgenden Funktionstasten:
  • Abbrechen: die aktuelle Partie wird sofort beendet, alle Daten dieser Partie gehen verloren.
  • Spiel beenden: die aktuelle Partie wird beendet, und der Wettkampf abgeschlossen (unterbrochen).
  • Weiter: der aktuelle Wettkampf wird fortgesetzt.
Durch Beendigung der App (Zurück-Taste) wird der Wettkampf so unterbrochen, dass bei einem Neustart der App der Wettkampf wieder automatisch fortgesetzt wird.

WICHTIGER HINWEIS
Bei manchen Geräten kann es zu einer Belastung bestimmter Komponenten (z.B. Prozessor, Bildschirm) kommen. Eine behutsame Anwendung der Computer-Match Funktionen ist erforderlich. Eine Haftung für Schäden wird nicht übernommen.

Analyse-Optionen

Im Analyse-Modus können bis zu vier Engines eingesetzt werden. Eine Analyse kann mit nur einer Engine (Berechnung der besten Varianten) als auch mit mehreren Engines (pro Engine wird die beste Variante angezeigt) erfolgen. Die Steuerung erfolgt mit der ersten Auswahl-Option.

Für jede Engine gibt es:
  • Auswahl-Taste: eine Liste der verfügbaren Engines wird angezeigt
  • Anzeige des Engine Namen: bei Berührung Aufruf der "Engine-Optionen"
Nach Betätigung von "Anwenden" wird die Analyse gestartet.
