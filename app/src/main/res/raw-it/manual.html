File PGN

Sono disponibili le seguenti funzioni:

  • Carica una partita
  • Salva, inserisci, sostituisci e cancella una partita
  • Scarica un file PGN dal Web
  • Copia una partita negli appunti
  • Copia una posizione negli appunti
  • Incolla una partita dagli appunti

Pressione prolungata per cancellare un file PGN (.pgn) e un file di database (.pgn-db).

Un file PGN può essere caricato anche da un gestore di file esterno

Database (.pgn-db)

L'applicazione usa un suo database proprietario (file: .pgn-db) per gestire le partite. Se il file di database non è presente, ne verrà creato uno nuovo quando si cercherà di caricare o di salvare una partita. Durante questo processo il file PGN risulterà bloccato. Tutte le altre azioni potranno essere eseguite senza problemi.

L'utilizzo del database PGN (.pgn-db) può essere disabilitato dalle impostazioni. Così facendo le partite saranno sempre aggiunte in coda al file PGN.

Motori scacchistici UCI con protocollo Open Exchange (OEX)

I motori Rodent e Stockfishsono integrati nell'applicazione. Nel Play Store sono disponibili altri motori scacchistici nel formato OEX (cerca: "chess engines oex"). Questi motori possono essere utilizzati all'interno di "Chess For All" e le loro opzioni gestite direttamente all'interno dell'app.

Libri delle aperture

ChessForAll include al suo interno un piccolo libro delle aperture. È tuttavia possibile usare altre risorse esterne nel formato "Polyglot" (.bin). La gestione dei libri delle aperture viene effettuata dalle impostazioni.
