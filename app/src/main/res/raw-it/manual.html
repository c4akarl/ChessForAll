File PGN

Sono disponibili le seguenti funzioni:

  • Carica una partita
  • Salva, inserisci, sostituisci e cancella una partita
  • Scarica un file PGN dal Web
  • Copia una partita negli appunti
  • Copia una posizione negli appunti
  • Incolla una partita dagli appunti

Pressione prolungata per cancellare un file PGN (.pgn) e un file di database (.pgn-db).

Un file PGN può essere caricato anche da un gestore di file esterno

Database (.pgn-db)

L'applicazione usa un suo database proprietario (file: .pgn-db) per gestire le partite. Se il file di database non è presente, ne verrà creato uno nuovo quando si cercherà di caricare o di salvare una partita. Durante questo processo il file PGN risulterà bloccato. Tutte le altre azioni potranno essere eseguite senza problemi.

L'utilizzo del database PGN (.pgn-db) può essere disabilitato dalle impostazioni. Così facendo le partite saranno sempre aggiunte in coda al file PGN.

Libri delle aperture

ChessForAll include al suo interno un piccolo libro delle aperture. È tuttavia possibile usare altre risorse esterne nel formato "Polyglot" (.bin). La gestione dei libri delle aperture viene effettuata dalle impostazioni.

Motori di scacchi

Richiamalo tramite il menu principale e seleziona "Motore scacchistico". Sono disponibili le seguenti funzioni:

Seleziona il motore di scacchi

La selezione determina il motore principale.
Questa app utilizza solo motori UCI forniti con il protocollo Open Exchange (OEX). Nel Play Store sono presenti app che forniscono motori scacchistici in formato OEX (cerca: "motori scacchistici oex").
Due motori, Rodent e Stockfish (che terminano con "CfA"), sono integrati in "Chess for All".

Opzioni del motore

Le opzioni specificate dallo sviluppatore del motore possono essere modificate per il motore selezionato con "Seleziona motore scacchistico". Nella finestra di dialogo di modifica è presente un pulsante "Informazioni" in basso a sinistra con una descrizione più dettagliata delle possibili attività. Le opzioni UCI vengono salvate e attivate prima dell'inizio di una nuova partita.

Computer Match

Questa opzione permette al motore scacchistico di giocare contro se stesso oppure contro un altro motore.
Sono disponibili le seguenti funzioni:
  •     Seleziona "Motore / Motore" o "Motore singolo"
  •     Pulsanti di selezione del motore "Bianco", "Nero" o "Motore"
  •     Seleziona "Posizione corrente". Se questa opzione non è attiva, ogni partita inizierà con la posizione base (Standard o Scacchi960). Se questa opzione è attiva, ogni partita inizierà con la posizione corrente della scacchiera.
  •     Seleziona "Alterna colore": nella modalità "Motore / Motore", il colore (bianco, nero) verrà cambiato dopo ogni partita.
  •     Seleziona "Salva partita". Se l'opzione è attiva ogni partita verrà salvata (percorso, file)
  •     Percorso, File: il file manager PGN viene richiamato con il pulsante "PGN". Qui è possibile gestire i file e selezionare anche i file esistenti (continuare il match tra computer).
  •     Event, Site: modifica dei campi di informazioni aggiuntive.
  •     Round: giochi completati
  •     Giochi: numero massimo di giochi. Quando l'ultima partita è finita, il match terminerà automaticamente.
  •     Visualizzazione dei dati della partita: nome dei motori e risultato
  •     Pulsante "Annulla": il processo verrà annullato
  •     Pulsante "Avvia / Continua": verrà avviata una nuova partita (round: 0) o continuata una interrotta.
Dopo aver premuto "Avvia / Continua", l'app passa alla "modalità match tra computer" e verranno visualizzati tutti i dati importanti della partita. Non è possibile richiamare ulteriori funzioni in questo stato. Verrà invece visualizzata una finestra di dialogo "Computer Match" con i seguenti tasti funzione:
  •     Annulla: la partita in corso verrà terminata immediatamente e tutti i dati di gioco andranno persi.
  •     Fine partita: la partita in corso verrà terminata e conclusa (interrotta).
  •     Continua: la partita in corso continuerà.
Chiudendo l'app (Pulsante Indietro) la partita verrà interrotta e riprenderà automaticamente al riavvio dell'app.

NOTA IMPORTANTE

In taluni dispositivi, alcuni componenti (ad es. Processore, schermo) possono essere soggetti a forte stress. È consigliato un uso oculato della funzione "Computer Match". Si declina ogni responsabilità per eventuali danni subiti.
