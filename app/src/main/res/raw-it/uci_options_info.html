Questo è un editor che può essere usato per modificare le opzioni UCI di un motore scacchistico.

Il motore fornisce dei valori predefiniti per ogni opzione, modificabili nei rispettivi campi. Il nome dell'opzione appare in giallo se il valore è quello predefinito, oppure in bianco se il valore è stato modificato. Se viene selezionato il nome dell'opzione viene ristabilito il suo valore predefinito.

Descrizione pulsanti:
Info: questa documentazione
Annulla: le modifiche andranno perse
Ripristina: tutte le opzioni verranno ripristinate al loro valore predefinito
OK: le modifiche verranno salvate e saranno effettive al successivo avvio del motore scacchistico

Alcuni motori scacchistici permettono di usare file aggiuntivi (es. nnue, weights) per incrementare la loro potenza di calcolo.
Se tra le opzioni disponibili è presente la voce "file", un file può essere aggiunto seguendo questa procedura:
1) utilizzando un gestore di file, copiare il file in una sottocartella (oppure crearne una nuova) di: Android / data / ccc.chess.gui.chessforall / files / engines
2) nel campo di modifica dell'opzione "file", selezionare il file precedentemente copiato
3) premere il pulsante OK

Per una descrizione dettagliata delle opzioni UCI consultare la documentazione del rispettivo motore scacchistico.
