This is an editor that can be used to change UCI options that have been defined by an engine.

The engine provides a default value for each option, which can be changed here in an edit field. The option name is displayed in either yellow (default) or white (change) text color. When you select the option name, a changed value is reset to the default value.

Button control:
Info: this documentation
Cancel: the saved data is retained
Reset: all options are reset to the default value
OK: the options are saved and will take effect the next time the engine is started

Some engines offer the option of using additional files (e.g. nnue, weights) to improve the playing strength of the engine.
If the option name contains the text "file", an additional file can be activated as follows:
1) with a file manager, copy the file in a subdirectory (or newly created directory) of: Android / data / ccc.chess.gui.chessforall / files / engines
2) select the previously copied file in the option's edit field
3) press the OK button

A detailed description of the UCI options can be found in the documentation of the respective chess engine.
