PGN files

The following functions are available:

  • Load a game
  • Save, insert, replace and delete a game
  • Download a PGN file
  • Game to clipboard
  • Position to clipboard
  • Game from the clipboard

Long press a PGN file to delete it (.pgn) and the database file (.pgn-db).

A PGN file can also be loaded from a file manager app.

Database (.pgn-db)

The program uses its own chess database (file: .pgn-db) to manage games. If there is no chess database file when loading/saving, it will be created. The status bar displays the current status of the database creation. During this process, the PGN file is locked. Other actions can be taken without any problems.

The use of the PGN database can be deactivated in the settings. In this mode, a game is always appended to the end of the file.

Opening books

A small opening book is already integrated in ChessForAll. However, external polyglot opening books (.bin) can also be used. The management of the opening books is done in the settings.

Chess engines

Call it up via the main menu and select "Chess Engine". The following functions are available:

Select chess engine

The selection determines the main engine.
This app only uses UCI engines that are provided with the Open Exchange (OEX) protocol. In the Play Store there are apps that provide chess engines in OEX format (search: "chess engines oex").
Two engines, Rodent and Stockfish (ending with "CfA"), are integrated in "Chess for All".

Engine options

The options specified by the engine developer can be changed for the engine selected with "Select Chess Engine". In the change dialog there is an "Info" button at the bottom left with a more detailed description of the possible activities. The UCI options are saved and activated before a new game starts.

Computer Match

With this option, multiple games, both in Single Engine and in Engine versus Engine mode, can run automatically.
The following functions are available:
  •     Select "Engine / Engine" or "Single Engine" (name of the main engine)
  •     Engine selection buttons "White", "Black" or "Engine"
  •     Select "Current game". If this option is not activated, each game will begin with the current basic position (Standard or Chess960). If this option is activated, each game begins with the current position on the board.
  •     Select "Change color": in the "Engine / Engine" mode, the color (white, black) is changed after each game.
  •     Select "Save game". When activated, each game is saved (path, file)
  •     Path, File: the PGN file manager is called up with the "PGN" button. Here you can manage files and also select existing files (continue computer match).
  •     Event, Site: Editing fields that can be changed.
  •     Round: games that have already been completed
  •     Games: maximum number of games. When the last game is over, the match ends automatically.
  •     Display of the match data: name of the engines and result
  •     "Cancel" button: the process is canceled
  •     "Start / Continue" button: a new match is started (round: 0) or an interrupted match is continued.
After pressing "Start / Continue", the app changes to "automatic match mode" and all important data of the match are displayed. No further functions can be called in this state. Instead, a Computer Match Dialog appears with the following function keys:
  •     Cancel: the current game is ended immediately, all data for this game is lost.
  •     Finish game: the current game is ended and the match is concluded (interrupted).
  •     Continue: the current match will be continued.
By closing the app (Back Button) the match is interrupted in such a way that the match is automatically continued again when the app is restarted.

IMPORTANT NOTE

With some devices, certain components (e.g. processor, screen) can be stressed. Careful use of the Computer Match functions is required. Liability for damage is not accepted.

Analysis Options

Up to four engines can be used in analysis mode. An analysis can be started with just one engine (calculation of the best variants) as well as with multiple engines (the best variant is displayed for each engine). The control takes place with the first selection option.

For each engine there are:
  • Select button: a list of the available engines is displayed
  • Display of the engine name: when touched, the "Engine Options" are called
The analysis is started after touching "Apply".
