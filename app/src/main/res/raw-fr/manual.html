Fichiers PGN

Les fonctions suivantes sont disponibles:

  • Charger le jeu
  • Enregistrer, insérer, remplacer et supprimer le lot
  • Télécharger le fichier PGN
  • Jeu >>> presse-papiers
  • Position >>> presse-papiers
  • Jeu du presse-papiers
Si vous appuyez longtemps sur un fichier PGN, ces derniers (.pgn) et le fichier de base de données (.pgn-db) sont supprimés.

Un fichier PGN peut également être chargé à partir d'un programme de gestionnaire de fichiers.

Base de données (.pgn-db)

Le programme utilise sa propre base de données d'échecs (fichier: .pgn-db) pour gérer les parties. S'il n'y a pas de fichier de base de données d'échecs lors du chargement / enregistrement, il sera créé. La barre d'état affiche l'état actuel de la création de la base de données. Au cours de ce processus, le fichier PGN est verrouillé. D'autres actions peuvent être prises sans aucun problème.

L'utilisation de la base de données PGN peut être désactivée dans les paramètres. En mode sauvegarde, un jeu est toujours ajouté à la fin du fichier.

Livres d'ouverture

Un petit livre d'ouverture est déjà intégré dans ChessForAll. Cependant, des livres d'ouverture polyglottes externes (.bin) peuvent également être utilisés. La gestion des livres d'ouverture se fait dans les paramètres.

Moteurs d'échecs

Appelez-le via le menu principal et sélectionnez "le moteur d'échecs". Les fonctions suivantes sont disponibles:

Sélectionnez le moteur d'échecs

La sélection détermine le moteur principal.
Cette application utilise uniquement les moteurs UCI fournis avec le protocole Open Exchange (OEX). Dans le Play Store, il existe des applications qui fournissent des moteurs d'échecs au format OEX (recherche: «moteurs d'échecs oex»).
Deux moteurs, Rodent et Stockfish (se terminant par "CfA"), sont intégrés dans "Chess for All".

Options de moteur

Les options spécifiées par le développeur du moteur peuvent être modifiées pour le moteur sélectionné avec "Select Chess Engine". Dans la boîte de dialogue de modification, il y a un bouton "Info" en bas à gauche avec une description plus détaillée des activités possibles. Les options UCI sont enregistrées et activées avant le début d'une nouvelle partie.

Match d'ordinateur

Avec cette option, plusieurs jeux, à la fois en mode Single Engine et en mode Engine versus Engine, peuvent s'exécuter automatiquement.
Les fonctions suivantes sont disponibles:
  •     Sélectionnez "Engine / Engine" ou "Single Engine" (nom du moteur principal)
  •     Boutons de sélection du moteur "Blanc", "Noir" ou "Moteur"
  •     Sélectionnez "Jeu actuel". Si cette option n'est pas activée, chaque partie commencera avec la position de base actuelle (Standard ou Chess960). Si cette option est activée, chaque partie commence par la position actuelle sur le plateau.
  •     Sélectionnez "Changer de couleur": dans le mode "Moteur / Moteur", la couleur (blanc, noir) est modifiée après chaque partie.
  •     Sélectionnez "Enregistrer la partie". Lorsqu'il est activé, chaque jeu est sauvegardé (chemin, fichier)
  •     Chemin, Fichier: le gestionnaire de fichiers PGN est appelé avec le bouton "PGN". Ici, vous pouvez gérer les fichiers et également sélectionner les fichiers existants (continuer la correspondance avec l'ordinateur).
  •     Événement, Site: Modification des champs qui peuvent être modifiés.
  •     Tour: parties déjà terminées
  •     Jeux: nombre maximum de jeux. Lorsque le dernier match est terminé, le match se termine automatiquement.
  •     Affichage des données de match: nom des moteurs et résultat
  •     Bouton "Annuler": le processus est annulé
  •     Bouton "Démarrer / Continuer": un nouveau match est lancé (tour: 0) ou un match interrompu se poursuit.
Après avoir appuyé sur "Démarrer / Continuer", l'application passe en "mode de match automatique" et toutes les données importantes du match sont affichées. Aucune autre fonction ne peut être appelée dans cet état. Au lieu de cela, une boîte de dialogue de correspondance d'ordinateur s'affiche avec les touches de fonction suivantes:
  •     Annuler: le jeu en cours se termine immédiatement, toutes les données de ce jeu sont perdues.
  •     Terminer le jeu: le jeu en cours est terminé et le match est terminé (interrompu).
  •     Continuer: le match en cours se poursuivra.
En fermant l'application (bouton Retour), le match est interrompu de manière à ce que le match se poursuive automatiquement lorsque l'application est redémarrée.

NOTE IMPORTANTE

Avec certains appareils, certains composants (par ex. Processeur, écran) peuvent être sollicités. Une utilisation prudente des fonctions Computer Match est requise. La responsabilité pour les dommages n'est pas acceptée.
