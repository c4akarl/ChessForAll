ChessForAll est un programme d'échecs avec le puissant moteur d'échecs Stockfish et de nombreuses fonctionnalités utiles.

Traits

  • Jeu contre ordinateur
  • Mode deux joueurs
  • Ordinateur contre ordinateur
  • Mode d'analyse
  • Mode d'édition
  • propre base de données (.pgn-db)
  • sauvegarder, insérer, remplacer et supprimer un jeu
  • Entrée de position
  • Presse-papiers (lot ou position)
  • Livre d'ouverture
  • Niveau de compétence
  • Options d'horloge d'échecs
  • Mode portrait / paysage

Mode d'emploi

Menu principal: balayez vers la droite depuis le bord gauche

Menu du jeu: glisser du bord droit vers la gauche

Menu des options du tableau: appui long sur l'échiquier

Options de conseil, essuyer:

  • prochain train: balayez vers la droite
  • train précédent: balayez vers la gauche
  • Position de départ: essuyer sur le bord gauche
  • Position finale: essuyer sur le bord droit

Rapide boutons de démarrage

  • Mode analyse / pause
  • Courir le train de l'ordinateur
  • Supprimer le train (de la position)
  • Changement de page
  • Train de retour
  • Avancer

Touches de démarrage rapide (glisser): charger un lot à partir du fichier PGN actuel

  • prochaine fête: à la bonne lingette
  • Jeu précédent: balayez vers la gauche
  • premier jeu: essuyer sur le bord gauche

Champ de presse Nom ou Elo: Modifier les données du jeu

Appuyez sur le temps du champ: changez l'heure actuellement; appui long: régler les options d'horloge d'échecs

Fichiers PGN

Les fonctions suivantes sont disponibles:

  • Charger le jeu
  • Enregistrer, insérer, remplacer et supprimer le lot
  • Télécharger le fichier PGN
  • Jeu >>> presse-papiers
  • Position >>> presse-papiers
  • Jeu du presse-papiers
Si vous appuyez longtemps sur un fichier PGN, ces derniers (.pgn) et le fichier de base de données (.pgn-db) sont supprimés.

Un fichier PGN peut également être chargé à partir d'un programme de gestionnaire de fichiers.

Base de données (.pgn-db)

Le programme utilise sa propre base de données d'échecs (fichier: .pgn-db) pour gérer les parties. S'il n'y a pas de fichier de base de données d'échecs lors du chargement / enregistrement, il sera créé. La barre d'état affiche l'état actuel de la création de la base de données. Au cours de ce processus, le fichier PGN est verrouillé. D'autres actions peuvent être prises sans aucun problème.

L'utilisation de la base de données PGN peut être désactivée dans les paramètres. En mode sauvegarde, un jeu est toujours ajouté à la fin du fichier.

Livres d'ouverture

Un petit livre d'ouverture est déjà intégré dans ChessForAll. Cependant, des livres d'ouverture polyglottes externes (.bin) peuvent également être utilisés. La gestion des livres d'ouverture se fait dans les options du jeu d'ordinateur, accessible via: Menu principal / Ordinateur ou en appuyant longuement sur le champ Calculs informatiques.

Paramètres Options utilisateur

  • Nom du joueur
  • Afficher la barre d'état
  • changer pour former après le train
  • Afficher la dernière position (chargement)
  • Désactiver le délai d'attente de l'écran
  • Navigation du jeu (carte)
  • Utiliser la base de données PGN
  • Effets sonores
  • Coordonnées  (1-h, 1-8)

Paramètres Stockfish

  • démarrage automatique de l'ordinateur à l'ouverture du programme
  • Afficher les calculs
  • 1er mouvement (blanc): mouvement aléatoire
  • Activer / désactiver le livre d'ouverture polyglotte
  • Livre d'ouverture Polyglot: charger un fichier .bin externe
  • Calculs informatiques: nombre de variantes et de mouvements
  • Niveau de compétence (0 ... 100)
  • Agressivité (0 ... 100), passivité (0 ...- 100)
  • Enregistrements de journal (messages de moteur)
