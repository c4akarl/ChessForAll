Il s'agit d'un éditeur qui peut être utilisé pour modifier les options UCI qui ont été définies par un moteur.

Le moteur fournit une valeur par défaut pour chaque option, qui peut être modifiée ici dans un champ d'édition. Le nom de l'option est affiché dans une couleur de texte jaune (par défaut) ou blanche (modification). Lorsque vous sélectionnez le nom de l'option, une valeur modifiée est réinitialisée à la valeur par défaut.

Contrôle du bouton:
Info: cette documentation
Annuler: les données enregistrées sont conservées
Réinitialiser: toutes les options sont réinitialisées à la valeur par défaut
OK: les options sont enregistrées et prendront effet au prochain démarrage du moteur

Certains moteurs offrent la possibilité d'utiliser des fichiers supplémentaires (par exemple nnue, poids) pour améliorer la force de jeu du moteur.
Si le nom de l'option contient le texte "file", un fichier supplémentaire peut être activé comme suit:
1) avec un gestionnaire de fichiers, copiez le fichier dans un sous-répertoire (ou répertoire nouvellement créé) de: Android / data / ccc.chess.gui.chessforall / files / engines
2) sélectionnez le fichier précédemment copié dans le champ d'édition de l'option
3) appuyez sur le bouton OK

Une description détaillée des options UCI peut être trouvée dans la documentation du moteur d'échecs respectif.
