PGN файлы

Доступны следующие функции:

  • Загрузка партии
  • Сохранение, вставка, замена и удаление партии
  • Загрузка файла PGN
  • Копирование партии в буфер обмена
  • Копирование позиции в буфер обмена
  • Вставка партии из буфера обмена

Долгое нажатие на файле PGN для его удаления (.pgn) и для удаленияфайла базы данных (.pgn-db).

Файл PGN может быть также загружениз менеджера файлов.

База данных партий (.pgn-db)

Для игры программа использует свою собственную шахматную базу данных (файл: .pgn-db). Если файл базы данныхпартий отсутствует при загрузке / сохранении, он будет создан. В строке состояния отображается текущий статус создания базы данных. Во время этого процесса файл PGN заблокирован. Другие действия могут быть предприняты без каких-либо проблем.

Использование базы данных PGN можно отключить в настройках. В этом режиме игра всегда добавляется в конец файла.

Книга дебютов

Небольшая книга дебютов уже интегрирована в ChessForAll. Однако, внешнюю более обширную книгу дебютов (.bin) также можно использовать. Управление открывающими книгами осуществляется в настройках.

Шахматные движки

Вызовите его через главное меню и выберите «Шахматный движок». Доступны следующие функции:

Выбрать шахматный движок

Выбор определяет главный двигатель.
Это приложение использует только механизмы UCI, которые предоставляются с протоколом Open Exchange (OEX). В Play Store есть приложения, которые предоставляют шахматные движки в формате OEX (поиск: «шахматные движки oex»).
В «Шахматы для всех» интегрированы два движка, Rodent и Stockfish (оканчивающиеся на CfA).

Варианты двигателя

Параметры, указанные разработчиком движка, могут быть изменены для движка, выбранного с помощью «Выбрать шахматный движок». В диалоговом окне изменения в левом нижнем углу есть кнопка «Информация» с более подробным описанием возможных действий. Параметры UCI сохраняются и активируются перед началом новой игры.

Компьютерный матч

С помощью этой опции несколько игр, как в режиме Single Engine, так и в режиме Engine vs. Engine, могут запускаться автоматически.
Доступны следующие функции:

  •     Выберите «Двигатель / Двигатель» или «Один двигатель» (название основного двигателя)
  •     Кнопки выбора двигателя «Белый», «Черный» или «Двигатель»
  •     Выберите «Текущая игра». Если эта опция не активирована, каждая игра будет начинаться с текущей базовой позиции (Standard или Chess960). Если эта опция активирована, каждая игра начинается с текущей позиции на доске.
  •     Выберите «Изменить цвет»: в режиме «Движок / Движок» цвет (белый, черный) меняется после каждой игры.
  •     Выберите «Сохранить игру». При активации каждая игра сохраняется (путь, файл)
  •     Путь, файл: файловый менеджер PGN вызывается кнопкой «PGN». Здесь вы можете управлять файлами, а также выбирать существующие файлы (продолжить сопоставление с компьютером).
  •     Событие, Сайт: поля редактирования, которые можно изменить.
  •     Раунд: игры, которые уже завершены
  •     Игры: максимальное количество игр. Когда последняя игра окончена, матч заканчивается автоматически.
  •     Отображение данных матча: название двигателей и результат
  •     Кнопка «Отменить»: процесс отменен
  •     Кнопка «Старт / Продолжить»: начинается новый матч (раунд: 0) или прерванный матч продолжается.

После нажатия «Старт / Продолжить» приложение переходит в «режим автоматического совпадения», и отображаются все важные данные матча. В этом состоянии нельзя вызывать другие функции. Вместо этого появляется диалоговое окно Computer Match со следующими функциональными клавишами:

  •     Отмена: текущая игра немедленно завершается, все данные для этой игры теряются.
  •     Завершить игру: текущая игра завершается, а матч завершается (прерывается).
  •     Продолжить: текущий матч будет продолжен.

При закрытии приложения (кнопка «Назад») матч прерывается таким образом, что матч автоматически продолжается снова при перезапуске приложения.

ВАЖНАЯ ЗАМЕТКА

В некоторых устройствах некоторые компоненты (например, процессор, экран) могут подвергаться нагрузке. Требуется осторожное использование функций Computer Match. Ответственность за ущерб не принимается.
