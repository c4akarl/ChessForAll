Instrucciones de uso

Menú principal: deslizar hacia la derecha desde el borde izquierdo

Menú del juego: desliza desde el borde derecho hacia la izquierda

Menú de opciones de la placa: pulsación larga en el tablero de ajedrez

Opciones de placa, limpiar:
  • próximo tren: desliza hacia la derecha
  • tren anterior: deslizar hacia la izquierda
  • Posición de inicio: limpie hasta el borde izquierdo
  • Posición final: limpie hasta el borde derecho

La opción de limpiar el tablero se puede desactivar en la configuración.

Botones de inicio rápido

  • Diálogo del juego
  • Iniciar / pausar computadora
  • Forzar el movimiento de la computadora
  • Eliminar tren (desde posición)
  • Cambio de página
  • Entrena de vuelta
  • Avanzar

Botones de inicio rápido (pulsación larga)

  • Configuraciones
  • Configuración de computadora
  • jugar automáticamente un juego (activar / desactivar)
  • Ejecutar movimiento de computadora
  • -
  • primer tren
  • último tren

Teclas de inicio rápido (deslizamiento): cargar un lote desde el archivo PGN actual

  • próxima fiesta: a la derecha
  • juego anterior: desliza hacia la izquierda
  • primer juego: limpiar al borde izquierdo
  • último juego: limpiar al borde derecho
  • Juego aleatorio: borrar de la última clave a la primera clave
  • Búsqueda de juegos: limpiar desde el primer botón hasta el último botón

Presione el campo Nombre o Elo: Editar datos del juego

Presione tiempo de campo: actualmente cambia el tiempo; pulsación larga: establece las opciones del reloj de ajedrez

Archivos PGN

Las siguientes funciones están disponibles:

  • Cargar juego
  • Guardar, insertar, reemplazar y eliminar lote
  • Descargar archivo PGN
  • Juego >>> portapapeles
  • Posición >>> Portapapeles
  • Juego del portapapeles
Al presionar un archivo PGN durante mucho tiempo se eliminan estos (.pgn) y el archivo de base de datos (.pgn-db).

Un archivo PGN también se puede cargar desde un programa administrador de archivos.

Base de datos (.pgn-db)

El programa usa su propia base de datos de ajedrez (archivo: .pgn-db) para administrar juegos. Si no hay un archivo de base de datos de ajedrez al cargar / guardar, se creará. La barra de estado muestra el estado actual de la creación de la base de datos. Durante este proceso, el archivo PGN está bloqueado. Se pueden tomar otras acciones sin ningún problema.

El uso de la base de datos PGN se puede desactivar en la configuración. En el modo guardar, siempre se agrega un juego al final del archivo.

La apertura de los libros

Un pequeño libro de apertura ya está integrado en ChessForAll. Sin embargo, también se pueden usar libros externos de apertura políglota (.bin). La gestión de los libros de apertura se realiza en la configuración.

Motores de ajedrez

Llámelo a través del menú principal y seleccione "Motor de Ajedreze". Están disponibles las siguientes funciones:

Seleccionar motor de ajedrez

La selección determina el motor principal.
Esta aplicación solo utiliza motores UCI que se proporcionan con el protocolo Open Exchange (OEX). En Play Store hay aplicaciones que proporcionan motores de ajedrez en formato OEX (búsqueda: "motores de ajedrez oex").
Dos motores, Rodent y Stockfish (que terminan con "CfA"), están integrados en "Chess for All".

Opciones de motor

Las opciones especificadas por el desarrollador del motor se pueden cambiar para el motor seleccionado con "Seleccionar motor de ajedrez". En el cuadro de diálogo de cambio hay un botón "Información" en la parte inferior izquierda con una descripción más detallada de las posibles actividades. Las opciones de UCI se guardan y se activan antes de que comience un nuevo juego.

Partido por computadora

Con esta opción, se pueden ejecutar automáticamente varios juegos, tanto en el modo de motor único como en el modo de motor contra motor.
Están disponibles las siguientes funciones:
  •     Seleccione "Motor / Motor" o "Motor único" (nombre del motor principal)
  •     Botones de selección de motor "Blanco", "Negro" o "Motor"
  •     Seleccione "Juego actual". Si esta opción no está activada, cada juego comenzará con la posición básica actual (Estándar o Chess960). Si esta opción está activada, cada juego comienza con la posición actual en el tablero.
  •     Selecciona "Cambiar color": en el modo "Motor / Motor", el color (blanco, negro) se cambia después de cada juego.
  •     Selecciona "Guardar juego". Cuando se activa, cada juego se guarda (ruta, archivo)
  •     Ruta, archivo: el administrador de archivos PGN se llama con el botón "PGN". Aquí puede administrar archivos y también seleccionar archivos existentes (continuar con la coincidencia de la computadora).
  •     Evento, Sitio: Editar campos que se pueden cambiar.
  •     Ronda: juegos que ya se completaron
  •     Juegos: número máximo de juegos. Cuando termina el último juego, el partido termina automáticamente.
  •     Visualización de los datos del partido: nombre de los motores y resultado
  •     Botón "Cancelar": el proceso se cancela
  •     Botón "Iniciar / Continuar": se inicia un nuevo partido (ronda: 0) o se continúa con un partido interrumpido.
Después de presionar "Iniciar / Continuar", la aplicación cambia al "modo de partido automático" y se muestran todos los datos importantes del partido. No se pueden llamar más funciones en este estado. En su lugar, aparece un cuadro de Diálogo Computer Match con las siguientes teclas de función:
  •     Cancelar: el juego actual finaliza inmediatamente, se pierden todos los datos de este juego.
  •     Finalizar juego: el juego actual finaliza y el partido concluye (se interrumpe).
  •     Continuar: el partido actual continuará.
Al cerrar la aplicación (Botón Atrás), el partido se interrumpe de tal manera que el partido continúa automáticamente cuando se reinicia la aplicación.

NOTA IMPORTANTE

Con algunos dispositivos, se pueden estresar ciertos componentes (por ejemplo, procesador, pantalla). Se requiere un uso cuidadoso de las funciones Computer Match. No se acepta responsabilidad por daños.

Opciones de análisis

Se pueden utilizar hasta cuatro motores en el modo de análisis. Se puede realizar un análisis con un solo motor (cálculo de las mejores variantes) así como con varios motores (se muestra la mejor variante para cada motor). El control se realiza con la primera opción de selección.

Para cada motor hay::
  • Botón Seleccionar: se muestra una lista de los motores disponibles
  • Visualización del nombre del motor: cuando se toca, se llaman las "Opciones del motor"
El análisis se inicia después de hacer clic en "Aplicar".
