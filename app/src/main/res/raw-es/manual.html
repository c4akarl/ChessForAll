Chess For All  2.32

ChessForAll es un programa de ajedrez con el poderoso motores y muchas características útiles.

Características

  • Juego contra la computadora
  • Modo de dos jugadores
  • Computadora contra computadora
  • Modo de análisis
  • Modo de edición
  • Chess960
  • Soporte PGN
  • propia base de datos (.pgn-db)
  • guardar, insertar, reemplazar y eliminar un juego
  • Entrada de posición
  • Portapapeles (lote o posición)
  • Motores UCI con protocolo Open Exchange (OEX)
  • Libro de apertura
  • Nivel de habilidad
  • Opciones de reloj de ajedrez
  • Modo retrato / paisaje

Instrucciones de uso

Menú principal: deslizar hacia la derecha desde el borde izquierdo

Menú del juego: desliza desde el borde derecho hacia la izquierda

Menú de opciones de la placa: pulsación larga en el tablero de ajedrez

Opciones de placa, limpiar:
  • próximo tren: desliza hacia la derecha
  • tren anterior: deslizar hacia la izquierda
  • Posición de inicio: limpie hasta el borde izquierdo
  • Posición final: limpie hasta el borde derecho

La opción de limpiar el tablero se puede desactivar en la configuración.

Botones de inicio rápido

  • Diálogo del juego
  • Iniciar / pausar computadora
  • Forzar el movimiento de la computadora
  • Eliminar tren (desde posición)
  • Cambio de página
  • Entrena de vuelta
  • Avanzar

Botones de inicio rápido (pulsación larga)

  • Configuraciones
  • Configuración de computadora
  • jugar automáticamente un juego (activar / desactivar)
  • Ejecutar movimiento de computadora
  • -
  • primer tren
  • último tren

Teclas de inicio rápido (deslizamiento): cargar un lote desde el archivo PGN actual

  • próxima fiesta: a la derecha
  • juego anterior: desliza hacia la izquierda
  • primer juego: limpiar al borde izquierdo
  • último juego: limpiar al borde derecho
  • Juego aleatorio: borrar de la última clave a la primera clave
  • Búsqueda de juegos: limpiar desde el primer botón hasta el último botón

Presione el campo Nombre o Elo: Editar datos del juego

Presione tiempo de campo: actualmente cambia el tiempo; pulsación larga: establece las opciones del reloj de ajedrez

Archivos PGN

Las siguientes funciones están disponibles:

  • Cargar juego
  • Guardar, insertar, reemplazar y eliminar lote
  • Descargar archivo PGN
  • Juego >>> portapapeles
  • Posición >>> Portapapeles
  • Juego del portapapeles
Al presionar un archivo PGN durante mucho tiempo se eliminan estos (.pgn) y el archivo de base de datos (.pgn-db).

Un archivo PGN también se puede cargar desde un programa administrador de archivos.

Base de datos (.pgn-db)

El programa usa su propia base de datos de ajedrez (archivo: .pgn-db) para administrar juegos. Si no hay un archivo de base de datos de ajedrez al cargar / guardar, se creará. La barra de estado muestra el estado actual de la creación de la base de datos. Durante este proceso, el archivo PGN está bloqueado. Se pueden tomar otras acciones sin ningún problema.

El uso de la base de datos PGN se puede desactivar en la configuración. En el modo guardar, siempre se agrega un juego al final del archivo.

Motores UCI con protocolo Open Exchange (OEX)

Los motores Rodent y Stockfish están integrados en la aplicación. Se pueden activar más motores en Play Store con la búsqueda "chess engine oex".

La apertura de los libros

Un pequeño libro de apertura ya está integrado en ChessForAll. Sin embargo, también se pueden usar libros externos de apertura políglota (.bin). La gestión de los libros de apertura se realiza en las opciones del juego de computadora, a las que se accede a través de: Menú principal / Computadora o presionando prolongadamente en el campo Cálculos de computadora.

Opciones de configuración de usuario

  • Nombre del jugador
  • Nombres de figuras
  • Mostrar barra de estado
  • cambio para entrenar después del tren
  • muestra la última posición (carga)
  • Desactivar el tiempo de espera de pantalla
  • Mostrar opciones de tren
  • Entrada de tren rápido
  • Navegación del juego (tablero)
  • Usar la base de datos PGN
  • Efectos de sonido
  • Coordenadas  (1-h, 1-8)
  • Modo de juego a ciegas

Opciones del motor

  • nicio automático de la computadora en la apertura del programa
  • Mostrar cálculos
  • Pensar siempre
  • 1er movimiento (blanco): movimiento aleatorio
  • Activar / desactivar libro de apertura políglota
  • Libro de apertura de Polyglot: carga el archivo .bin externo
  • Cálculos de computadora: número de variantes, movimientos y líneas mostradas.
  • Nivel de habilidad (0 ... 100)
  • Agresividad (0 ... 100), pasividad (0 ...- 100)
  • Información de depuración
  • Registros de registro (mensajes del motor)
