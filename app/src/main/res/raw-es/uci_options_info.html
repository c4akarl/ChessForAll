Este es un editor que puede usarse para cambiar las opciones de UCI que han sido definidas por un motor.

El motor proporciona un valor predeterminado para cada opción, que se puede cambiar aquí en un campo de edición. El nombre de la opción se muestra en color de texto amarillo (predeterminado) o blanco (cambiar). Cuando selecciona el nombre de la opción, un valor modificado se restablece al valor predeterminado.

Control de botones:
Info: esta documentación
Cancelar: los datos guardados se retienen
Restablecer: todas las opciones se restablecen al valor predeterminado
OK: las opciones se guardan y entrarán en vigor la próxima vez que se arranque el motor.

Algunos motores ofrecen la opción de utilizar archivos adicionales (por ejemplo, nnue, pesos) para mejorar la fuerza de reproducción del motor.
Si el nombre de la opción contiene el texto "file", se puede activar un archivo adicional de la siguiente manera:
1) con un administrador de archivos, copie el archivo en un subdirectorio (o directorio recién creado) de: Android / data / ccc.chess.gui.chessforall / files / engines
2) seleccione el archivo previamente copiado en el campo de edición de la opción
3) presione el botón OK

Se puede encontrar una descripción detallada de las opciones de la UCI en la documentación del motor de ajedrez respectivo.
